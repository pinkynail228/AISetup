# Definition of Done — Phase 3: Copywriting

Every box checked before closing the phase and moving to Phase 4.

- [ ] Every page listed in `site/sitemap.md` has a corresponding copy file at `site/copy/<slug>.md` written from `process/templates/copy-page.template.md`.
- [ ] Every copy file's slug and brief reference match the corresponding `site/page-briefs/<slug>.md` — no mismatches.
- [ ] Every page has exactly one H1 heading, and its text matches (or is a direct expression of) the primary message stated in that page's brief.
- [ ] Heading hierarchy is valid on every page: H1 appears before any H2, H2 appears before any H3, no heading levels are skipped.
- [ ] Every section defined in the page brief's section outline has corresponding copy in the copy file — no outlined section is left blank or marked "TBD".
- [ ] Every primary CTA from the page brief is present in the copy file with exact button/link text written out.
- [ ] All microcopy blocks are complete: every button label, form field label, placeholder, error message, and empty state relevant to the page is written — none are left as "button text here" or similar.
- [ ] 404 page copy exists at `site/copy/404.md` — heading, one short body line, and a link back to home — even though the 404 page is not listed in the sitemap.
- [ ] No copy file contains lorem ipsum, "TBD", "placeholder", or any dummy text anywhere.
- [ ] The tone and vocabulary throughout all copy files is consistent with the tone of voice adjectives and free of the banned words/phrases defined in `site/strategy.md`.
- [ ] Copy is written in the site language declared in `site/strategy.md` — not in any other language.
- [ ] The copy file status field is set to `draft` (not left blank).
- [ ] `STATE.md` is updated: Phase 3 status set to `done`, completion date recorded.
