# Сетапы

Каждая папка здесь — один сетап; его манифест, версия и changelog — в `SETUP.yaml`.

Правила обновления — в корневом [`AGENTS.md`](../AGENTS.md): изменение сетапа = PR с бампом версии и записью в changelog. После merge в `main` CI автоматически публикует релиз `id@version` с zip-архивом и пересобирает витрину.
